# <PersonalInfo values change nextStep previousStep />

The Personal Info component that is renderd on the second step of the Registration component. This component can be skipped if the user signing up is a student.

PropTypes:
Name Type Required Description Default
values object Yes The form field values passed down from the Register component
change func Yes The change function to handle the input changes in the form and update the state in the parent component
nextStep func Yes The next step function to handle progressing further in the form
previousStep func Yes The prevous step function to handle backwards movement in the form
